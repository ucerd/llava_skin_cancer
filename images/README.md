# Source images

Original dermoscopic images are not redistributed in this repository.

The study used a 500-image subset of DERM12345:

- Article DOI: `10.1038/s41597-024-04104-3`
- Dataset DOI: `10.7910/DVN/DAXZ7P`
- Upstream license: CC BY 4.0

For a complete reproduction, place the selected 500 source images in this directory using the local filenames listed in `../src/data/split_manifest.csv` (`001.jpg` through `500.jpg`).

The authoritative source-to-local filename mapping is not present in this release. The source image, patient, and lesion identifiers therefore cannot be restored reliably from the available local metadata alone.
