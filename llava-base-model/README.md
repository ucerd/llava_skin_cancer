# LLaVA base model

Third-party foundation-model weights are not redistributed in this repository.
Download `llava-hf/llava-1.5-7b-hf` into this directory before training or inference:

```bash
hf download llava-hf/llava-1.5-7b-hf --local-dir llava-base-model
```

Use of the model remains subject to its upstream license and terms.
